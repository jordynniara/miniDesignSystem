Default responsive navigation bar. Typically place at top of screen. Can use position: sticky so it stays at top of screen while scrolling. Decrease screen size to see responsivity (menu toggle)
Code shown is for card on display. CSS can be modified as needed using classes .navbar, .navbar-brand, .navbar-togger (menu toggle button), , .navbar-nav (item list), and .nav-item (items). See Bootstraps navbar documentation for details.

Coloring for the navigation are chosen to fit the theme of the system while also maintaining WCAG color contrast standards. Links and Buttons should be used distinctly from each as defined in Buttons. Items in navigation bar should be focusable, and have the appropriate aria labelling for keyboard navigation and screen readers. See Mozilla Web Documents for more accessibility guidelines.
